bpython is a fancy interface to the Python interpreter for Linux, BSD, OS X and Windows (with some work). bpython is released under the MIT License. It has the following (special) features:

  1. In-line syntax highlighting
  2. Readline-like autocomplete with suggestions displayed as you type.
  3. Expected parameter list for any Python function.
  4. "Rewind" function to pop the last line of code from memory and re-evaluate.
  5. Send the code you've entered off to a pastebin.
  6. Save the code you've entered to a file.
  7. Auto-indentation.

I contributed to resolving GitHub issues on the bpython project page. Some of these issues included:

  • Entering an endless loop through symbolic links when the program is parsing through the computer files to import modules for autocomplete
  • Improving the import completion with background imports of modules to offer more detailed options
  • Other smaller issues such as adding instructions for external editors and writing tests

Curtsies is a Python 2.6+ & 3.3+ compatible library for interacting with the terminal. It uses FmtStr objects to display strings with colors and styles in a terminal with ANSI escape sequences. The above gif is a Tron game by Jack Rybarczyk made with Curtsies.

I investigated GitHub isues on the Curtsies project page. Some of these issues included:

  • Writing more detailed error messages with visualizations that cover some more edge cases
  • Other smaller issues such as making the readme work with PyPi, reformatting the program and adding more key combinations
