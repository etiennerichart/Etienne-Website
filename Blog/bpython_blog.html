bpython

Try it out:

Try out bpython by typing 'bpython' out into the terminal and then pressing Enter. bpython lets you learn and explore python and its libraries by proposing autocomplete options which avoid spending time researching as well as an undo function that avoid retyping code (Control + R on Mac).
