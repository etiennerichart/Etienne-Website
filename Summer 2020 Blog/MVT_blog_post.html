Visualizing Music to the Terminal

The Idea:

I first came up with this idea while learning about ANSI escape codes and trying to write a program that would move the terminal cursor around to create things like a loading bar. I thought of using the terminal as a canvas to draw and update a picture that would represent the sound file playing along.



If you are interested in learning more about the terminal and ANSI escape codes, I recomend taking a look at these resources:

  • Terminal Whispering
  • Build your own Command Line with ANSI escape codes
  • Everything you never wanted to know about ANSI escape codes

Creating the Canvas

The first step in my project was to create a working canvas that would display and update correctly in the terminal. I started out by creating a program that would display a 80x24 rectangle that changes colors randomly a hundred times.

To do this I used the u"\u001b[{n}A" escape code that moves the cursor up n rows, the u"\u001b[{n}D" that moves the cursor left n columns, and u"\u001b[{30-37}m" code that sets the text color to one of 8 different colors. I also finished the code with u"\u001b[0m" that resets the color to default.

Some useful ANSI Escape Codes

Color ANSI Esc Code Cursor Direction ANSI Esc Code
Black \u001b[30m Up{n} \u001b[{n}A
Red \u001b[31m Down{n} \u001b[{n}B
Green \u001b[32m Right{n} \u001b[{n}C
Yellow \u001b[33m Left{n} \u001b[{n}D
Blue \u001b[34m
Magenta \u001b[35m
Cyan \u001b[36m
White \u001b[37m
Reset \u001b[0m

Adjusting the Canvas to the Terminal

To make sure the image in viewable I needed the canvas to adjust to the size of the terminal. To do this I had to find a way to get the terminal's size.

A terminal is made of a certain amount of rows and columns. Each cell holding one character (as long as they are of standard width). After some research I found an easy way to get the number of rows and columns here. Using this code which returns the rows and columns as a String:

import os
rows, columns = os.popen('stty size', 'r').read().split()
